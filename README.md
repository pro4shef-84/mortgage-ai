# AI Mortgage Processing Assistant

This repository contains an AI-native workflow system for mortgage document processing.

The goal is to automate document collection, validation, borrower reminders, and officer review preparation for independent mortgage loan officers.

## Core Rule

LLM is NOT the workflow engine.

Deterministic services control:
- state transitions
- reminders
- escalations
- audit logs

AI assists with:
- document classification
- borrower messaging
- summaries
